
# E-commerce RENT

Tienda destinada a la venta de peliculas. Ecommerce creado para 
coderhouse


## Instalacion

Instalar el proyecto con npm

```bash
  npm install my-app
  cd my-app
```
    
## Como correrlo localmente

Clonar el proyecto

```bash
  git clone https://github.com/AdolfoaR/E-commerceCH.git
```

Ir al directorio del proyecto

```bash
  cd my-app
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  npm run start
```
